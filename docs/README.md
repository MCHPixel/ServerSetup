# Documentation

This folder contains the main guide documents for the ServerSetup repository.

- `tutorial.md` — comparisons of Minecraft server software and key plugins
- `proxy-server.md` — proxy overview for BungeeCord, Velocity, and Waterfall
- `templates.md` — template usage notes and recommendations
- `glossary.md` — definitions of common Minecraft server and proxy terms

For step-by-step installation instructions, see the `setup/` folder.

## Recommended flow

1. Read `tutorial.md` to decide on server software and plugins.
2. Read `proxy-server.md` if you plan to use a proxy.
3. Read `templates.md` to understand the starter files.
4. Use `setup/README.md` to choose the right installation guide.
