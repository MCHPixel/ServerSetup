Please note that you will need to change plugins to your own preferences when using our templates.

In all of them, we are using Paper by default due to performance, compatibility, and the additional configuration options described in `docs/tutorial.md`.

These templates do not include proxy setup, so if you want to use proxies, please look at `docs/proxy-server.md` and `../setup/README.md`.

## Template folders

- `templates/linux/` — Linux-specific starter files
- `templates/windows/` — Windows-specific starter files
- `templates/example/` — base example files to customize
