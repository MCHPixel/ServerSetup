# Glossary

## Bukkit
A Minecraft server platform that is easy to use and supports Bukkit plugins. It is older and less actively developed than Spigot or Paper.

## Spigot
A fork of Bukkit with performance improvements and strong compatibility with Bukkit plugins.

## Paper
A high-performance fork of Spigot that includes extra optimizations and configuration options.

## Proxy
A server layer that connects multiple Minecraft servers into a network so players can switch between them without reconnecting.

## BungeeCord
A popular Minecraft proxy solution used to connect multiple servers in a network.

## Velocity
A newer Minecraft proxy focused on performance, security, and reliability.

## Waterfall
A fork of BungeeCord with performance and security improvements. It is less actively maintained than the other two proxies.

## Plugins
Add-on code that extends Minecraft server behavior, like permissions, chat formatting, and world tools.
