# Templates

This folder contains starter templates for creating a Minecraft server environment.

## Template folders

- `templates/linux/` — Linux-specific startup files and configuration examples
- `templates/windows/` — Windows-specific startup files and configuration examples
- `templates/example/` — Base example template files you can customize for any setup

## Notes

- The templates are designed for Paper by default for best performance and compatibility.
- You will still need to update plugin lists, server properties, and any proxy settings for your own server.
- These templates do not include proxy setup. If you want to connect multiple servers, see `../setup/README.md`.
