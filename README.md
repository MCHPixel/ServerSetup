# ServerSetup Guide

This repository provides a structured Minecraft server and proxy setup guide with template files and step-by-step instructions.

## Quick links

- `docs/README.md` — overview of the documentation section
- `docs/tutorial.md` — compare server software and plugin recommendations
- `docs/proxy-server.md` — proxy overview and guidance
- `docs/templates.md` — how to use the starter templates
- `docs/glossary.md` — terms and definitions for Minecraft servers and proxies
- `setup/README.md` — how to use the setup guides

## Contents

- `docs/` — guide documents for server software, proxy setup, templates, and glossary
- `setup/` — installation/setup instructions for servers and proxies
- `templates/` — ready-to-use starter templates for Linux, Windows, and example files

## Recommended reading order

1. `docs/tutorial.md` — choose the right server software and plugins
2. `docs/proxy-server.md` — choose the right proxy option
3. `docs/templates.md` — learn how to use the starter templates
4. `setup/README.md` — find the right setup guide
5. `setup/server/` or `setup/proxy/` — follow the installation steps

## Notes

- Use `templates/linux/` or `templates/windows/` for OS-specific starter files.
- `templates/example/` includes a base template you can customize.
- Proxy configuration is handled separately in `setup/proxy/`.
