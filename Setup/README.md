# Setup Guides

This folder contains step-by-step setup instructions for Minecraft server software and proxy servers.

## Server setup

- `server/bukkit.md` — Bukkit server installation and setup
- `server/spigot.md` — Spigot server installation and setup
- `server/paper.md` — Paper server installation and setup

## Proxy setup

- `proxy/bungeecord.md` — BungeeCord proxy installation and setup
- `proxy/velocity.md` — Velocity proxy installation and setup
- `proxy/waterfall.md` — Waterfall proxy installation and setup

## How to use these guides

1. Read `../docs/tutorial.md` to decide which server software best fits your needs.
2. Read `../docs/proxy-server.md` if you plan to connect multiple servers with a proxy.
3. Follow the appropriate guide from `server/` or `proxy/`.
4. Use the `../templates/` folder for starter files and examples.
