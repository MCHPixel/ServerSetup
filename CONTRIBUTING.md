# Contributing

Thanks for checking out ServerSetup! If you want to help improve this guide, please follow these steps:

1. Open an issue for new setup guides, corrections, or template improvements.
2. Submit a pull request with clear changes and updated documentation.
3. Keep file and folder names consistent with the existing structure.
4. Use Markdown formatting and short, clear headings.

## Suggested improvements

- Add more server or proxy setup guides
- Improve plugin recommendations and configuration examples
- Add local screenshots or diagrams
- Correct typos and improve grammar
